📌 Sobre o projeto

Este projeto é um site informativo dedicado à Grécia, explorando sua rica história, cultura, turismo e gastronomia. O objetivo é apresentar conteúdos relevantes de forma visual e acessível para quem deseja conhecer mais sobre esse país fascinante.

🌍 Funcionalidades
Página inicial com visão geral da Grécia
Seção sobre história e mitologia
Guia turístico com principais destinos
Informações sobre cultura e tradições
Galeria de imagens
Design responsivo para dispositivos móveis
🛠️ Tecnologias utilizadas
HTML5
CSS3
Frameworks como Bootstrap
